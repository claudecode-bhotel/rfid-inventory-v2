DataWedge設定ガイド

Zebraデバイスで以下のDataWedgeプロファイルを設定してください:

  1. DataWedgeアプリを開く
  2. 新しいプロファイルを作成（名前: RFID在庫管理）
  3. 関連付けアプリ → ブラウザ（Chrome / Enterprise Browser）を選択
  4. Input → RFID Input → 有効化
  5. Output → Keystroke Output → 有効化
  6. キーストローク間遅延 → 50ms
  7. データ末尾アクション → Enter（改行）
